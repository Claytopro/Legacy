Ackermann Non-REcursive algorithm

How to run:
compile program with  
gnatmake -Wall ackermann.adb

run with 2 command line arguments that are integers
if any is invalid or to many values entered then program will not run

valid example:

./ackermann 1 1 


invalud example:

./ackermann 1.2 1
or
./ackermann 1 1 3
or
./ackermann 1 c 4
